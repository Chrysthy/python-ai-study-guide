<br>


## 🧩 Context

This project was developed as part of the **Accenture - Python for Data Analysis and Automation** bootcamp, offered by DIO.

The goal was to explore the **Python programming language** using NotebookLM as a learning support tool, organizing knowledge from multiple sources and applying AI-assisted study techniques.

<br>


## 🎯 Learning Objectives

- Understand Python fundamentals  
- Explore from basic concepts to more advanced structures  
- Use AI as a support tool in the learning process  
- Develop organization and knowledge synthesis skills  

<br>


## 📚 Source Curation

- Official Python documentation  
- Tutorials from **freeCodeCamp**  
- Curso em Vídeo (Python)  
- Materials about IDEs such as **PyCharm**  

<br>


### 💡 Principais aprendizados | Key Learnings

- Clear prompts improve response quality  
- AI should support learning, not replace it  
- Iterative refinement is essential
